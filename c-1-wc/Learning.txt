Summary of Differences
Memory Allocation: & creates a pointer by obtaining the memory address of a variable. It doesn't allocate new memory but creates a reference to existing memory.
Dereferencing: * is used to dereference a pointer, allowing you to access or change the value at the pointer's memory location.
